## Why PostgreSQL on VPS?

1. **FREE** - No additional database costs
2. **Fast** - Same server = low latency
3. **Simple** - One server to manage
4. **Scalable** - Can upgrade VPS when needed
5. **Cost-effective** - Cheapest option

---

## When to Use Managed Database Instead?

Consider managed database (Neon/Supabase) if:
- You exceed 5GB database size
- You want automatic backups
- You prefer not to manage database
- You need automatic scaling

**Cost**: Add ~£15-20/month
